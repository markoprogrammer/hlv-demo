# HLV Demo

Rebuild of [hooloovoo.rs](https://hooloovoo.rs) — demo projekat za AI Show & Grow prezentaciju.

**Live:** https://hlv-demo.vercel.app

## Stack

- React Router v7 (SSR, framework mode)
- Tailwind CSS v4
- TypeScript
- Bun
- Vercel

## Commands

```bash
bun run dev        # localhost:5173
bun run build      # production build
bun run typecheck  # type check
```
